Chart JS

Chart JS is een JavaScript bibliotheek waarmee je grafieken en diagrammen kunt maken.

Op de website van Chart JS kun je heel veel informatie vinden over hoe je zo een grafiek of diagram kunt aanmaken. Zie bijvoorbeeld Chart JS voor de 'getting started'.

Hiernaast zie je een voorbeeld van een grafiek met het aantal studenten per klas (aantallen kloppen niet met de werkelijkheid).

De opdracht:

Hieronder staan 2 canvas elementen. Eén voor opdracht 1 en één voor opdracht 2.

Voor opdracht 1 ga je de json fetchen uit het mapje json met de naam 'assignment1.json'. De data uit deze json ga je afbeelden in een grafiek op het canvas met de class 'chart-assignment-1'. De grafiek laat het aantal JavaScript ontwikkelaars zien over de jaren 2017 tot 2021.

Voor opdracht 2 ga je de json fetchen uit het mapje json met de naam 'assignment2.json'. De data uit deze json ga je afbeelden in een grafiek op het canvas met de class 'chart-assignment-2'. De grafiek laat het aantal ontwikkelaars zien van diverse programmeertalen over de jaren 2017 tot 2021.

Beide opdrachten uitgewerkt? Heel goed. Voeg nog een grafiek of diagram toe met data uit een zelfgekozen API. Probeer ook te experimenteren met de verschillende chart types zoals 'pie', 'doughnut', 'bar', 'bubble', etc..
